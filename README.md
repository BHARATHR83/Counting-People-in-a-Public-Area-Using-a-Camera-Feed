

# AI Crowd Management System

This project is a real-time crowd monitoring system using YOLOv8 and DeepSORT.
It detects, tracks, and analyzes people in video streams and shows results
on a Streamlit dashboard with alerts and heatmaps.

## Features
- Real-time person detection
- People counting and tracking
- Overcrowding alerts
- Heatmap visualization
- Firebase logging
- Admin & User login

## Technologies Used
- Python
- Streamlit
- YOLOv8
- DeepSORT
- OpenCV
- Firebase

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run main.py
